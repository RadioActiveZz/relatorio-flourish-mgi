Recursos para Desenvolvedores (API)

Esta seção detalha como a API do Flourish pode ser utilizada pela equipe do MGI para integração com o ColaboraHub, garantindo dados em tempo real e segurança.

Automação e Dados em Tempo Real

A API permite a criação e atualização programática de visualizações. Para a integração com o ColaboraHub, a equipe pode automatizar o processo de atualização de gráficos com dados de sistemas internos, garantindo que o público tenha acesso a informações sempre frescas.

# Exemplo de uso da API em Python
import requests
def update_chart(api_key, chart_id, new_data):
    url = f"https://api.flourish.studio/v1/visualisations/{chart_id}"
    headers = {"Authorization": f"Token {api_key}"}
    payload = {"data": new_data}
    response = requests.put(url, headers=headers, json=payload)
    return response.json()

Simulação de Chamada Segura à API

Esta simulação demonstra a chamada segura da API do Flourish a partir do frontend. Na prática, a requisição seria enviada para um endpoint no servidor do MGI, que por sua vez, faria a chamada real à API com a chave de segurança.
